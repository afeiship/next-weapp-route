# next-weapp-route
> Route for weapp


## usage:
```js
import NxWeappRoute from 'next-weapp-route';

// common apis:
NxWeappRoute.go()
NxWeappRoute.replace()
NxWeappRoute.back()

// special for weapp:
NxWeappRoute.reLanuch()
NxWeappRoute.switchTab()
```
